# GridSync: Number Assembly

* Press Space/Enter to place the falling pieces.
* Press ESC to pause the game.
* Between levels 1-5, you have 3 lives. After 6 until 8, you have 2 lives. And at 8th ad 9th levels, you have 1 life.
* Falling speed of pieces increases after each level.
* The score depends on accuracy of syncing the pieces into target digits.
* Avoid system hazards falling. Do not press any key during system hazards fall.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
